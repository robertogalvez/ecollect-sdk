# Changelog

All notable changes to the ecollect SDK monorepo will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0-alpha.1] - 2026-05-31

### Added
- TypeScript/JavaScript SDK Alpha with full ecollect API coverage (103 tests)
- PHP SDK with WooCommerce and PrestaShop plugins (66 tests)
- Kotlin Android SDK with coroutines and OkHttp
- Swift iOS SDK with async/await and CryptoKit (zero dependencies)
- Sandbox module with 10 pre-configured test scenarios
- GitHub Actions CI for all 4 SDKs
- Monorepo root structure with npm workspaces
- CONTRIBUTING guide and SECURITY policy
